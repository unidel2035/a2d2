# Contributing to A2D2

Thank you for your interest in contributing to A2D2!

Please see our detailed [Contributing Guidelines](docs/phase-8-deployment/technical/contributing.md) for:

- Development setup
- Code style guidelines
- Testing requirements
- Pull request process
- Community guidelines

## Quick Start

```bash
# Clone and setup
git clone https://github.com/unidel2035/a2d2.git
cd a2d2
bundle install
rails db:setup

# Run tests
rails test

# Check code style
bundle exec rubocop
```

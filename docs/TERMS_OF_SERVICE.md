## 1. Общие положения

Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует отношения между Оператором платформы A2D2 (далее — «Платформа», «Сервис») и пользователем (далее — «Пользователь», «Вы») при использовании Платформы.

### 1.1 Принятие соглашения

- Используя Платформу, Вы принимаете условия настоящего Соглашения в полном объеме
- Если Вы не согласны с условиями, Вы должны прекратить использование Платформы
- Регистрация на Платформе означает полное и безоговорочное принятие Соглашения

### 1.2 Изменение соглашения

- Оператор вправе вносить изменения в Соглашение в одностороннем порядке

- Продолжение использования Платформы после изменений означает согласие с ними

## 2. Описание сервиса

### 2.1 Назначение платформы

Платформа A2D2 предназначена для:
- Автоматизации бизнес-процессов с использованием ИИ-агентов
- Управления документами и данными
- Оркестрации интеллектуальных агентов
- Управления роботизированными системами
- Анализа и отчетности

### 2.2 Доступность сервиса

- Оператор стремится обеспечить доступность Сервиса 99.5% времени
- Плановые технические работы проводятся с предварительным уведомлением
- Оператор не несет ответственности за временную недоступность по техническим причинам

## 3. Регистрация и учетная запись

### 3.1 Требования к регистрации

Для регистрации требуется:
- Достижение 18 лет (или действующая от лица юридического лица)
- Предоставление достоверной информации
- Согласие на обработку персональных данных
- Принятие условий настоящего Соглашения

### 3.2 Безопасность учетной записи

Пользователь обязуется:
- Обеспечивать конфиденциальность пароля
- Использовать надежные пароли (минимум 8 символов, включая заглавные, строчные, цифры, спецсимволы)
- Включить многофакторную аутентификацию (MFA) при работе с критичными данными
- Немедленно уведомить Оператора о любом несанкционированном доступе

### 3.3 Ответственность за учетную запись

- Пользователь несет ответственность за все действия, совершенные с его учетной записи
- При подозрении на компрометацию учетной записи необходимо немедленно изменить пароль

## 4. Правила использования

### 4.1 Разрешенное использование

Пользователь вправе:
- Использовать Платформу для законных целей
- Создавать и обрабатывать данные в рамках предоставленных прав доступа
- Интегрировать Платформу с другими системами через API

### 4.2 Запрещенное использование

Пользователю запрещается:
- Нарушать законодательство Российской Федерации
- Загружать вредоносное ПО или вирусы
- Пытаться получить несанкционированный доступ к системе
- Проводить действия, направленные на нарушение работы Платформы
- Копировать, модифицировать или распространять программное обеспечение Платформы
- Использовать автоматизированные средства для сбора данных (кроме официального API)
- Выдавать себя за другого пользователя или организацию
- Передавать доступ к учетной записи третьим лицам

### 4.3 Ограничения использования

- **Rate Limiting**: API ограничен 300 запросами в 5 минут на IP-адрес
- **API Token**: 1000 запросов в час на токен
- **Попытки входа**: максимум 5 попыток на email в течение 20 минут
- **Блокировка**: учетная запись блокируется после 5 неудачных попыток входа

## 6. Конфиденциальность и безопасность

### 6.1 Обработка персональных данных

Обработка персональных данных осуществляется в соответствии с:
- [Политикой конфиденциальности](PRIVACY_POLICY.md)
- ФЗ-152 «О персональных данных»
- ГОСТ Р ИСО/МЭК 27001-2021

### 6.2 Меры безопасности

Платформа реализует:
- **Шифрование**: TLS 1.3 для передачи, AES-256 для хранения
- **Аутентификация**: MFA с TOTP
- **Контроль доступа**: RBAC (Role-Based Access Control)
- **Аудит**: журналирование всех критичных операций (срок хранения 3 года)
- **Защита от атак**: rate limiting, CSRF protection, XSS protection
